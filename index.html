Installer sur Android

Pour avoir cette app sur votre écran d'accueil :

1. Ouvrez ce fichier dans Chrome

2. Appuyez sur le menu ⋮ en haut à droite

3. Sélectionnez "Ajouter à l'écran d'accueil"

4. Confirmez l'installation

L'app fonctionne entièrement hors ligne. Vos données restent sur votre téléphone.
